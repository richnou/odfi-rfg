    • tsc 0x28

      This register implements the EXTOLL time stamp counter (TSC). The counter is incremented at every clock cycle. A software write to the register loads the counter with the given value. In addition, the counter can be set from the global interrupt hardware to implement a globally sychronized time source.

      Scala API Path

          var tsc = read("info_rf/tsc")
    write(value) into "info_rf/tsc"
